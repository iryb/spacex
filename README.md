# SpaceX
React app for booking space trips.

## Features
- Fetchinng graphql data from SpaceX API
- Favorite trips

## Tech Stack
- Atomic design pattern
- Storybook
- Typescript
- React Apollo
- SpaceX API (graphql)
- Recoil
- Styled Components
- Webpack
